Element Selectors

One of the most basic ways to select "something" to style is to target the HTML element directly. One thing to remember is here is that this approach will style every instance of this element in an HTML document.

Class Selectors

To target something more specifically, we can use class selectors. Class selector don't have to be associated with any specific element.

ID Selectors

We can also use ID selectors, which can only be used once per document. This makes them less reusable for styling, and makes classes more preferred.

Combinators

Combinators combine other selectors.

Adjacent Sibling

Selects an element that's next to another (specified) element. For example, selecting the paragraph element directly after the H3 element.

General Sibling

Selects all instances of an element that comes after another (specified) element even if they're not adjacent to each other.
